# React JS, React Native, NodeJS e MongoDB

## Proof of Concept - utilizando de forma resumida:

### Front-end 
  * Web - React.js
  * Mobile - React Native
  
### Back-end
  * NodeJS
  * MongoDB.

### Conceitos
  * Websocket - atualizações em tempo real
  * Roteamento - react-router-dom
  * State - utilização de estado
  * Estilização - CSS
  * Componentização
  * ES6+
  * Propriedades e Eventos
